So how where they added on to the menu.
in the registry for internet explorer there is key called MenuExt.
it let's you add Menu items linked to files on your Computer.

from install.log
RegDB Key: Software\Microsoft\Internet Explorer\MenuExt\Zoom &In
RegDB Val: C:\WINDOWS\WEB\zoomin.htm
RegDB Root: 1
RegDB Key: Software\Microsoft\Internet Explorer\MenuExt\Zoom &In
RegDB Val: 02
RegDB Name: contexts
RegDB Root: 1
RegDB Key: Software\Microsoft\Internet Explorer\MenuExt\Zoom &Out
RegDB Val: C:\WINDOWS\WEB\zoomout.htm
RegDB Root: 1
RegDB Key: Software\Microsoft\Internet Explorer\MenuExt\Zoom &Out
RegDB Val: 02
RegDB Name: contexts
RegDB Root: 1

In the key MenuExt\ there is a key called Zoom &In\ this key is the name that will show
up on the menu list.
at MenuExt\Zoom &In key there is to value items one binary and one string value.

contexts Tells with right-click menu to show item on Example : 01 = Everywhere 02 = on images
(Default)  Tells with file to activate if item is clicked Example c:\windows\web\ZoomIn.htm

Now it's time to try ZoomIn.htm And ZoomOut.htm
To start with, you need to add some values to the windows registry
(if you have no clue about what windows registry is Please follow the instructions)

1.  Start Regedit. (the windows registry editor)
2.  Open HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\MenuExt\
3.  Add A new key Called Zoom &In.
4.  Open Zoom &In
5.  Add new String value and name it contexts
6.  Change the value of contexts to 02
7.  Now Change the value of  (Default) to The Path
     Where you unzipped the file And ZoomIn.htm  Exampel : C:\downloads\Vbscript\ZoomIn.htm

8.  Now it's time to add the second item on the menu list.
9.  Open HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\MenuExt\
10. Add A new key Called Zoom &Out.
11. Open Zoom &Out
12. Add new String value and name it contexts
13. Change the value of contexts to 02
14. Now Change the value of  (Default) to The Path
      Where you unzipped the file And ZoomOut.htm  Example : C:\downloads\Vbscript\ZoomOut.htm

If Internet Explorer is running you must restart it (it Loads information from registry on startup)
now open any web page that has pictures on it.

right-click on a picture and click on Zoom In. The picture will get larger
right-click on a picture and click on Zoom Out. And the picture will get smaller
